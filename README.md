Objetivo:
O objetivo deste projeto é permitir que os alunos compreendam e apliquem os conceitos de Knowledge Graphs, incluindo a criação de uma base de conhecimento, a implementação de código para manipulação de grafos e a análise das potenciais aplicações e limitações dessa abordagem.

Descrição:
Os alunos devem estudar sobre Knowledge Graphs, que são estruturas de dados que representam conhecimento de uma forma semântica, ligando entidades e conceitos através de relacionamentos. Eles devem criar uma base de conhecimento em forma de grafo, implementar código para manipular esse grafo (adicionar, remover, consultar nós e relacionamentos) e escrever um relatório descrevendo a aplicação, a base de conhecimento criada, os potenciais de aplicação e as limitações dessa abordagem.
